Council decision: auto-summarize commits to obsidian. Option A: per-commit summaries. Every git commit generates obsidian/Memory/Stories/commits/{date}-{hash}.md with structured frontmatter (type, date, author, files_changed, tags). Also writes to SQLite for agent retrieval. Git post-commit hook runs commit-summarize.sh. Rationale: agent memory value wins, granular episodic data essential for causal analysis.
